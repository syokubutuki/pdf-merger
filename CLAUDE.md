# PDF結合ツール

## プロジェクト概要
複数のPDFを1つに結合するWebアプリ。
登録不要・広告なし・シンプルな1ページ構成。

## 技術構成
- Next.js（App Router）
- TypeScript
- Tailwind CSS
- pdf-lib（PDF結合処理）

## 特徴
- ブラウザ上だけで処理が完結（サーバーにファイルを送信しない）
- ファイルの順番を並び替え可能
- 複数ファイルを一括選択可能
